# DHT Sensor
A simple library for using DHT sensors with Arduino module (DHT11, DHT22)
---
## Sensors properties
| Properties | DHT11 | DHT22 (AM2302) |
| - | :-: | :-: |
| Voltage (V) | 3..5 | 3..5 |
| Current (mA) | 2.5 | 2.5 |
| Temp (&deg;C) | 0..50 | -40..80 |
| Temp tolerance (&deg;C) | 2 | 0.5 |
| Hum (%) | 20..80 | 0..100 |
| Hum tolerance (%) | 5 | 2..5 |
| Query frequence (Hz) | 1 | 0.5 |
